PUFFERFISH INSTALLATION INSTRUCTIONS


FIRST STEPS:
1. Delete any previous versions of Pufferfish you have installed.
2. Right click the Pufferfish2-8.zip file > go to Properties > If you see a check box in the bottom right that says "Unblock", check it and hit Apply > OK.
3. Unzip the Pufferfish2-8.zip file.

RHINO 5 NEXT STEPS:
1. You should be using atleast Rhino 5 SR12.
2. Open Grasshopper and go to File > Special Folders > Components Folder (The folder it actually opens will be called LIBRARIES, not Components, DO NOT PUT PUFFERFISH IN THE FOLDER ACTUALLY CALLED COMPONENTS).
3. Place the Pufferfish2-8.gha file in that Libraries folder.
4. Completely close both Grasshopper and Rhino fully. 
5. Launch Rhino, then Grasshopper, you should now have a Pufferfish tab in Grasshopper. 

RHINO 6 NEXT STEPS:
1. If you have Rhino 5 also, just install the Pufferfish2-8.gha in Rhino 5 (follow Rhino 5 instructions above), Rhino 6 will pick it up automatically from there.
2. Do not install the Pufferfish2-8.gha in both Rhino 5 and Rhino 6. 
3. If you just have Rhino 6 then, Open Grasshopper and go to File > Special Folders > Components Folder (The folder it actually opens will be called LIBRARIES, not Components, DO NOT PUT PUFFERFISH IN THE FOLDER ACTUALLY CALLED COMPONENTS).
4. Place the Pufferfish2-8.gha file in that Libraries folder.
5. Completely close both Grasshopper and Rhino fully. 
6. Launch Rhino, then Grasshopper, you should now have a Pufferfish tab in Grasshopper.

RHINO 7 WIP NEXT STEPS:
1. If you have Rhino 5 or 6 also, just install the Pufferfish2-8.gha in Rhino 5 or 6 (follow Rhino 5 or 6 instructions above), Rhino 7 WIP will pick it up automatically from there.
2. Do not install the Pufferfish2-8.gha in more than one Rhino version. 
3. If you just have Rhino 6 and Rhino 7 WIP then, Open Grasshopper in Rhino 7 WIP and go to File > Special Folders > Components Folder (The folder it actually opens will be called LIBRARIES, not Components, DO NOT PUT PUFFERFISH IN THE FOLDER ACTUALLY CALLED COMPONENTS).
4. Place the Pufferfish2-8.gha file in that Libraries folder.
5. For installing the Pufferfish WIP SubD Components, Open Grasshopper in Rhino 7 WIP and go to File > Special Folders > User Objects Folder, place the files from the "Pufferfish WIP SubD Components" folder into that User Objects Folder.
6. Pufferfish WIP SubD Components REQUIRE PUFFERFISH2-8.gha TO BE INSTALLED TO WORK
7. Completely close both Grasshopper and Rhino fully. 
8. Launch Rhino, then Grasshopper, you should now have a Pufferfish tab in Grasshopper.

ADDITIONAL:
1. If you have followed the directions above and still have issues installing Pufferfish please try restarting your computer.
2. If even then you have an installation problem double check that the folder you have the Pufferfish2-8.gha in is called "Libraries" not "Components".
3. If even then you have an installation problem then please email me at ekimroyrp@gmail.com with screenshots and detailed information about what you are seeing.
